记忆要点

  • 使用前需要打开流，即.stream()
  • 执行收集操作是.collect()
  • 收集操作支持：Collectors.toList()，Collectors.toSet()，Collectors.toMap()，Collectors.joining()
  • 执行过滤的操作是.filter()
  • 执行集合操作的是.map()
  • 声明操作有两种写法，一种是Object::Function，另外一种是o -> o.function()
